Hey there! Welcome to my portfolio website. :)


I made this website primarily as a way to showcase my skills in HTML, CSS and JS as well as the numerous projects I've worked on. I also wanted to see what aspects I struggled on.


In the creation of this website, I made sure all standard practices were followed, including but not limited to:

  • Responsive design that works with all screen sizes
  • Ability to enlarge images on click/tap
  • Use of accessibility tags
  • Clean, structured code with comments
  • Readable and maintainable to ensure easy addition and removal of features
  • Files laid out in a simple and organised way
  • Use of local version control software

The code for this website can be seen at this GitHub repository.
